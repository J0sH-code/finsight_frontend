# FinSight Frontend

> The frontend prototype for FinSight, an AI-powered expense and budget tracking platform.

## Overview

FinSight is a personal finance application designed to help users track expenses, manage budgets, and eventually receive AI-powered financial insights.

This repository contains the frontend application built with React, TypeScript, and Vite. It currently serves as the UI foundation and design prototype while backend development continues in a separate private repository.

## Current Status

🚧 **Active Development**

The frontend currently consists of generated and manually refined UI screens that demonstrate the intended user experience and product direction.

Most views currently use placeholder data while integration with backend services is ongoing.

The backend currently includes:

* JWT Authentication
* User Management
* PostgreSQL Database Integration
* Budget Management APIs
* Service Layer Architecture
* Global Exception Handling
* Core Domain Models

Frontend integration with these services is currently in progress.

## Vision

FinSight aims to become more than a simple expense tracker.

The long-term goal is to provide:

* Expense and income tracking
* Monthly budget management
* Financial analytics and reporting
* AI-generated spending insights
* Personalized budgeting recommendations

## Project Goals

FinSight was created as a learning project to gain hands-on experience with:

* Backend Development
* Authentication & Security
* Database Design
* REST API Development
* Frontend Development
* System Architecture
* AI Integration

Rather than building isolated tutorials, the goal is to create a complete end-to-end application that solves a real problem while exposing the challenges involved in building production-style software.
